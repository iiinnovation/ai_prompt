1
平台支持扩展 高优先级
新增主流平台适配
当前仅支持 5 个平台，以下高频平台缺失：
Claude.ai Perplexity Grok (x.com) 豆包 文心一言 讯飞星火 Copilot POE
通用输入框适配方案
当前为硬编码平台列表，建议增加 通用模式：检测页面中的 contenteditable / textarea 元素，非 AI 平台也能直接注入，而不只是复制到剪贴板。
2
模板库丰富度 高优先级
缺失场景补充
  • 产品/运营场景（PRD、需求文档、竞品分析）
  • 数据分析场景（数据解读、报表描述）
  • 图像生成 Prompt（Midjourney / DALL·E 格式）
  • 角色扮演 / 模拟对话
  • 社交媒体文案（小红书、微信公众号风格）
模板社区共享
允许用户一键将自己的模板发布到公共模板库，其他用户可按评分/分类浏览并导入，实现社区沉淀。
3
AI 能力增强 中优先级
Prompt 一键优化
集成 API（可选配），对用户自己写的粗糙 Prompt 自动进行结构化改写，提升指令清晰度和有效性。
根据页面上下文推荐
识别当前页面内容（如用户在 GitHub 上看代码，则优先推荐"代码审查""Bug 排查"等模板），实现智能召回。
Prompt 效果打分
使用后对 AI 的回复质量打分（1-5 星），系统根据评分数据自动优化模板推荐权重。
4
数据同步与多端 中优先级
Chrome 账号同步
使用 chrome.storage.sync 替代 chrome.storage.local，登录同一 Chrome 账号的多台设备自动同步模板数据，无需手动导出导入。
第三方云备份
支持一键备份到 Notion 数据库 / Google Drive / GitHub Gist，满足高级用户的数据归档需求。
5
交互体验优化 低优先级
面板内联编辑
在弹出面板中直接双击模板文本即可编辑，不必每次跳转到选项页，减少操作路径。
变量预设默认值
模板变量支持设置默认值，如 {{语言|中文}}，用户若不修改可直接回车使用，加速常见场景注入。
Prompt 历史记录
记录最近注入的 20 条 Prompt（含变量填写内容），支持快速重复使用上次填写过的组合。
6
